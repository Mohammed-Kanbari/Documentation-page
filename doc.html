HTML and CSS

HTML stands for HyperText Markup Language and is used to structure content on the web.

CSS (Cascading Style Sheets) is used to style HTML content, including layout, colors, fonts, and more.

HTML example: <div>Hello, World!</div> CSS example: div { color: blue; }
  • HTML is the structure of a webpage.
  • CSS is the styling of a webpage.
  • CSS can be inline, internal, or external.
  • HTML tags are case-insensitive.
  • CSS allows responsive design via media queries.
JavaScript and Java

JavaScript is a dynamic programming language used for web development.

Java is a static, object-oriented programming language used for building complex applications.

JavaScript example: let x = 10; Java example: int x = 10;
  • JavaScript is used for front-end web development.
  • Java is used for backend applications.
  • JavaScript is interpreted, Java is compiled.
  • JavaScript runs in browsers, Java runs in virtual machines.
  • Java is known for platform independence.
Python Programming

Python is an interpreted, high-level programming language used for web development, data analysis, and more.

Python example: print("Hello, World!")
  • Python supports multiple programming paradigms.
  • It is known for its simplicity and readability.
  • Python is widely used in data science.
  • It has a large community and rich library ecosystem.
  • Python can be used for both small and large-scale applications.
Data Structures

Data structures are ways of organizing and storing data to perform operations efficiently.

Array example: let arr = [1, 2, 3];
  • Arrays store elements of the same type.
  • Linked lists consist of nodes where each node points to the next.
  • Stacks follow Last In, First Out (LIFO) order.
  • Queues follow First In, First Out (FIFO) order.
  • Hash tables store key-value pairs for fast lookup.
Algorithms

Algorithms are step-by-step procedures for solving problems and performing computations.

Sorting example: let arr = [3, 1, 2]; arr.sort();
  • Sorting algorithms organize data into a specific order.
  • Search algorithms help find elements in data structures.
  • Recursion is a technique where a function calls itself.
  • Divide and conquer is a method of solving problems by breaking them down.
  • Dynamic programming is used to solve problems by breaking them into subproblems.
